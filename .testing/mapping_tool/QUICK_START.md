# 🚀 快速开始指南

## 运行映射分析工具

### 方法一：直接运行主程序

```bash
cd .testing/mapping_tool
python RUN_THIS_TOOL.py
```

### 方法二：性能测试对比

```bash
cd .testing/mapping_tool  
python performance_comparison.py
```

## 输出文件说明

运行完成后会生成以下文件：

1. **comprehensive_analysis_report.json** - 完整的分析数据(JSON格式)
2. **device_mapping_analysis_report.md** - 人类可读的分析报告(Markdown格式)
